これは何？

Mozilla Developer Network（通称、MDN）のコンテンツは誰でも編集する事が出来ます。MDN のコンテンツマネジメントシステムは過去に二回変更されており、その度に文書のフォーマットが変更されました。いくつかのページではこの変更に追従できておらず、システムエラーや文書の破綻が発生しています。

このツールは、MDN の旧システム「MediaWiki」、「DekiWiki」のコードのいくつかを新システム「Kuma」のフォーマットに変換し、迅速な復旧及び編集作業を実現するものです。エラーの発生するテンプレートの修正やコメントアウト、不正な HTML を出力するマークアップの修正、URL の修正、Mozilla 製品名などのスペル修正などをワンクリックで行う事が出来ます。

また初回翻訳時にこのツールで英語のページの文書をコンバートする事により、URL を日本語ページ用に変換し、一部の翻訳も自動で行う事が出来ます。更に翻訳済ページのよくある間違いも修正します。

Advanced mode にチェックが入った状態の場合、まだ製作者が完全にコントロールできていない置換操作も行います。後述の「問題点」の項をご覧ください。

このツールはリクエストをいただいて製作したものですが、未完成です。しかし度重なる変更に継続的に対応しており、実際にこれを用いて、のべ 10000 ページ以上を修正していますが、概ね動作は良好です(｀・ω・)b
